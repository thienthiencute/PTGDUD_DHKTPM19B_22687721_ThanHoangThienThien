# 📚 PTGDUD – Phát Triển Giao Diện Ứng Dụng

Repository này chứa các bài tập và project thực hành trong môn **Phát Triển Giao Diện Ứng Dụng (PTGDUD)**.
Mục tiêu của môn học là giúp sinh viên nắm vững cách xây dựng giao diện web hiện đại bằng **ReactJS**, từ các khái niệm cơ bản đến các kỹ thuật tối ưu hiệu năng.

---

# 🎯 Mục tiêu môn học

* Hiểu cách xây dựng **Single Page Application (SPA)**
* Làm việc với **ReactJS**
* Sử dụng **Component-based Architecture**
* Quản lý state bằng **React Hooks**
* Xây dựng UI tương tác
* Tối ưu hiệu năng ứng dụng React
* Kết nối API và xử lý dữ liệu
